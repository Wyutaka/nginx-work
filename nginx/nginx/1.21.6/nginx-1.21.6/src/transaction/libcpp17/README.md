# libcpp17
C++17 and C++20 standard library implementation for C++11 or more.

# features
+ std::any (cpp17::any)
+ std::optional (cpp17::optional)
+ std::string_view (cpp17::string_view)
  + std::basic_string_view
+ std::span (cpp17::span)
+ not to use RTTI
+ can use in C++11 or more versions
